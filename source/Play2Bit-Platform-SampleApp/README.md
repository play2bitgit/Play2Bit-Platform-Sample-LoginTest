Bora SKD Sample Project

- 로그인과 지갑 정보 얻어오는 구성이 샘플로 작업되어 있습니다.
- 유니티 버젼은 2018.4.6f1 LTS 버젼입니다.

Project Info

- GreeWebView를 이용한 로그인 및 지갑 정보를 받아오는 로직을 샘플로 만들었습니다.
    = GreeWebView는 dist폴더에서 패키지를 설치하시면 됩니다.(이 프로젝트에는 적용되어 있습니다.)

- 현재 모바일만 작업되어 있습니다.(안드로이드는 확인 완료, IOS는 미정)

- 추가되는 내용이 있으면 계속 업데이트될 예정입니다.
